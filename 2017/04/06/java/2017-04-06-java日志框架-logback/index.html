简单介绍

logback当前分成三个模块：logback-core,logback- classic和logback-access。

  1. logback-core 是其它两个模块的基础模块。
  2. logback-classic是log4j的一个 改良版本。此外logback-classic完整实现SLF4J API。
  3. logback-access提供HTTP访问日志功能。
